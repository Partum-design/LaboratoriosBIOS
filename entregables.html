Plataforma · bios-platform

Estructura base del sistema entregada al inicio del proyecto. Define cómo está organizada la plataforma, qué tecnologías se usan y cómo se conectan las distintas partes del sistema.

Herramientas que usamos para construir tu sistema
🌐
Framework web · Next.js 14
Es el motor principal del sitio. Permite que cada pantalla cargue rápido, se vea bien en cualquier dispositivo y esté bien posicionada en Google. Es la misma tecnología que usan empresas como TikTok y Nike.
🗄️
Base de datos y autenticación · Supabase
Aquí se guardan de forma segura todos los datos: resultados de estudios, información de pacientes y empresas, y folios. También gestiona quién puede entrar a cada parte del sistema según su rol.
🔗
Integración de facturación · SICOFI
Conexión directa con el sistema de facturación electrónica certificado ante el SAT. Las empresas con convenio pueden generar y descargar sus CFDI automáticamente desde el portal, sin procesos manuales.
🎨
Diseño y estilos · Tailwind CSS
Sistema de diseño que permite construir interfaces consistentes y atractivas en mucho menos tiempo. Garantiza que el sitio se vea uniforme en todos los dispositivos: escritorio, tableta y celular.
Cómo está organizado el sistema
🏥
Portal del Paciente
Acceso con folio + fecha de nacimiento · Ver, descargar y compartir resultados · Rastreo de estudio en tiempo real
🏢
Portal de Empresa B2B
Acceso con RFC · Gestión de empleados · Facturación SICOFI integrada · Reportes de consumo
⚙️
Panel de Administración Interna
Solo equipo BIOS · Carga de estudios · Gestión de usuarios · Métricas y reportes del sistema
💡 ¿Por qué estas herramientas? La combinación Next.js + Supabase es actualmente el estándar de la industria para aplicaciones de este tipo: segura, rápida de desarrollar y fácil de mantener. Permite crecer el sistema con nuevas funciones sin tener que reconstruirlo desde cero.
